Use OpenVINO JavaScript API for your Node.js application.

Usage
#####################

1. Install the **openvino-node** package:

   .. code-block::

      npm install openvino-node

2. Use the **openvino-node** package:

   .. code-block::

      const { addon: ov } = require('openvino-node');
